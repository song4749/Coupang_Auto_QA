직렬 설치 시 설치 비용 가이드

1. 세탁기만 신규 구매해서 기존 건조기와 직렬설치하는 경우


- 기존 사용 중인 건조기와 기존 사용 중인 키트(또는 앵글)를 사용하여 직렬설치하는 경우,
올리기 비용과 재설치 비용이 발생합니다.
※ 기존 건조기의 올리기 및 재설치가 필요하므로 올리기 비용과 재설치 비용 별도 발생

2. 건조기만 신규 구매하여 기존 세탁기와 직렬설치하는 경우


- 기존 사용 중인 세탁기와 기존 사용 중인 키트(또는 앵글)를 사용하여 직렬설치하는 경우,
건조기 올리기 비용이 발생합니다.
- 기존 사용 중인 세탁기와 신규로 구매한 키트(또는 앵글)를 사용하여 직렬설치하는 경우,
별도 비용이 발생하지 않습니다.


※ 직렬설치 시 기존 사용 중인 세탁기 위치를 변경하는 경우, 세탁기 재설치 비용이 발생합니다.

3. 세탁기와 건조기 모두 신규 구매하여 직렬설치하는 경우


- 키트(또는 앵글)도 신규로 구매하여 직렬설치하는 경우, 별도 비용이 발생하지 않습니다.
- 보유 중인 키트(또는 앵글)로 직렬설치하는 경우, 건조기 올리기 비용이 발생합니다.

4. 세탁기와 건조기의 직렬설치를 희망하는 경우 해당 제품과 호환되는 전용 상단 설치 키트
또는 전용 앵글(설치기사 문의)이 필요하며, 전용 키트 또는 전용 앵글 외 타 제품은 설치가 불가합니다.

5. 신규로 구매한 키트를 사용하여 건조기 설치 시, 키트(또는 앵글)와 건조기는 동일 날짜에 함께 배송받아
세탁기와 직렬설치를 해야 건조기 올리기 비용이 발생하지 않습니다.
